# CityListen Tulcán

CityListen Tulcán - La ciudad que escucha a sus ciudadanos

## 🚀 **Subir a GitHub y Render.com - Pasos Exactos**

Como Git no está en el PATH, te daré las instrucciones manuales exactas:

### **Opción 1: Usar GitHub Desktop (Más Fácil)**

1. **Descarga e instala GitHub Desktop** desde: https://desktop.github.com/
2. **Abre GitHub Desktop**
3. **File → Add Local Repository**
4. **Selecciona la carpeta:** `C:\Users\User\CascadeProjects\citylisten`
5. **Current branch:** "main"
6. **Repository name:** "citylisten-tulcan"
7. **Click:** "Publish repository"
8. **Selecciona:** "Public"
9. **Publish repository**

### **Opción 2: Subir Manual por Web (Sin Git CLI)**

1. **Ve a:** https://github.com/new
2. **Repository name:** `citylisten-tulcan`
3. **Selecciona:** "Public"
4. **Click:** "Create repository"
5. **Click:** "Add file" → "Upload files"
6. **Arrastra estos archivos:**
   - `index.html`
   - `styles.css`
   - `script.js`
   - `App.jsx`
   - `index.jsx`
   - `package.json`
   - `vite.config.js`
   - `Procfile`
   - `index-react.html`
7. **Commit changes:** "Initial commit - CityListen Tulcán"
8. **Click:** "Commit changes"
9. **Click:** "Confirm upload"

---

## 🌐 **Una Vez en GitHub - Conectar con Render**

1. **Ve a:** https://render.com
2. **Inicia sesión** con tu cuenta GitHub
3. **Click:** "New +" → "Web Service"
4. **Busca y selecciona:** `citylisten-tulcan`
5. **Render detectará automáticamente** el proyecto
6. **Build Command:** `npm install`
7. **Start Command:** `npm start`
8. **Node Version:** 18
9. **Click:** "Create Web Service"
10. **Espera 2-3 minutos**
