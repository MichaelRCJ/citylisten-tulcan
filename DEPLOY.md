# CityListen Tulcán - Plataforma de Reporte Ciudadano

## 🚀 **Despliegue Rápido via GitHub Pages**

### **Opción 1: Subida Manual (Recomendada)**

1. **Ve a [GitHub](https://github.com)**
2. **Crea un nuevo repositorio** llamado `citylisten-tulcan`
3. **Sube estos archivos:**
   - `index.html`
   - `styles.css` 
   - `script.js`
   - `README.md`

4. **Activa GitHub Pages:**
   - Ve a Settings → Pages
   - Selecciona fuente: "Deploy from a branch"
   - Elige rama: `main`
   - **¡Listo!** Tu app estará en: `https://[tu-usuario].github.io/citylisten-tulcan/`

### **Opción 2: Netlify Drop (Más Rápida)**

1. **Ve a [Netlify Drop](https://app.netlify.com/drop)**
2. **Arrastra esta carpeta completa** al navegador
3. **Netlify creará un link automático** en segundos

### **Opción 3: Vercel (Alternativa Profesional)**

1. **Ve a [Vercel](https://vercel.com)**
2. **Importa tu repositorio de GitHub**
3. **Despliegue automático** con link personalizado
