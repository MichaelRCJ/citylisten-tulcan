# 🚀 CityListen Tulcán - Listo para Vercel

## 📋 **Pasos para Subir a Vercel (30 segundos)**

### **Opción 1: Subida Web (Más Fácil)**

1. **Abre:** https://vercel.com
2. **Inicia sesión** con GitHub/GitLab/Email
3. **Click:** "Add New..." → "Project"
4. **Selecciona:** "Browse Git Repository"
5. **Click:** "Link Git Repository"
6. **Pega:** `https://github.com/tu-usuario/citylisten-tulcan`
   *(o crea un repo nuevo y pega ese link)*
7. **Click:** "Continue" → "Deploy"
8. **¡Espera 30 segundos!** Vercel creará tu link

### **Opción 2: Subida Manual (Sin Git)**

1. **Crea cuenta:** https://vercel.com
2. **Descarga:** Vercel CLI: `npm i -g vercel`
3. **En terminal:** `cd C:\Users\User\CascadeProjects\citylisten`
4. **Ejecuta:** `vercel --prod`
5. **Sigue las instrucciones** en pantalla
